REGISTER
POST / auth / register 
BODY firstName, lastName, emailOrMobile, password , confirmPassword
RESPONSE
201 {{ accessToken : String }, user: { id:Number , firstName: String}}
400 { message : String, emailExist:true }
500 { message : String }
